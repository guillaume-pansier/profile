Technologies used for the website

UI libraries, frameworks and tools

Angular 6

The base framework. At the time of the creation, the most recent Angular version.

Angular Material

A UI library offering many standard components, or how not to reinvent the wheel.

Bootstrap

Famous HTML, CSS, and JS library. Used here mainly for the css helper classes and utilities.

Angular Animation

Side library of Angular which provides the illusion of motion. Based on the CSS animations.

On the development side

Karma, Jasmine

All classes are unit tested with the tools coming from the Angular CLI (Karma, Jasmine).

Protractor + Selenium

End to end Protractor tests have been configured and are ran with Selenium after each commit.

Docker, Jenkins

Used for the CI/CD pipeline.
